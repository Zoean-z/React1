# 学伴 - 学习任务提醒应用

一个基于 React + Vite 构建的学习任务提醒应用，帮助学生组建学习小队，互相督促完成学习任务。

## 🎯 项目特色

- **智能提醒系统**：根据学生历史活跃时段，智能推送学习提醒
- **小队协作**：3-5人组建学习小队，互相督促共同进步
- **轻提醒机制**：戳一戳、发送鼓励、AI队长提醒等温和提醒方式
- **数据可视化**：运营Dashboard展示关键指标，策略演示展示智能提醒原理

## 📱 页面功能

### 1. 首页 (`/`)
- **统计卡片**：今日进度、剩余任务、连续打卡天数、今日目标
- **今日任务面板**：显示待完成的学习任务（数学、英语、物理、语文等）
- **学习小队面板**：展示小队信息和成员完成状态
- **智能提醒面板**：解释为什么在当前时间推送提醒
- **奖励栏**：显示学习能量和连续打卡奖励

### 2. 小队页面 (`/team`)
- **小队统计**：今日完成率、连续打卡天数、当前排名
- **成员列表**：显示每个成员的任务完成状态和进度
- **轻提醒工具**：戳一戳、发送鼓励、AI队长提醒
- **提醒小贴士**：使用提醒的注意事项

### 3. 加入小队页面 (`/join`)
- **好友组队**：邀请好友创建小队
- **系统匹配**：智能匹配学习节奏相近的同学
- **小队规则**：3-5人小队、只看任务状态、提醒次数限制、隐私保护
- **冷启动说明**：新组队阶段(前7天) vs 稳定组队阶段(7天后)
- **常见问题**：FAQ解答

### 4. 消息页面 (`/messages`)
- **消息列表**：小队消息、AI队长消息、系统通知
- **聊天界面**：群聊功能，支持发送消息
- **快捷功能**：提醒成员、发送鼓励、AI队长提醒
- **小队状态**：实时显示小队完成率和连胜天数

### 5. 运营Dashboard (`/dashboard`)
- **关键指标**：今日DAU、任务学生数、未完成用户数、小队平均完成率
- **智能提醒打开率**：展示提醒效果
- **用户分层占比**：自驱型、被动型、流失风险型用户分布
- **完成率提升曲线**：对比使用/未使用智能提醒的效果

### 6. 策略演示页面 (`/strategy`)
- **学生画像**：展示单个学生的分层结果
- **历史活跃时段**：热力图展示学生学习时间分布
- **当前分层**：被动型用户识别
- **推荐策略**：系统Push、队友戳一戳、AI队长提醒、短信升级、微信回流

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173 查看效果

### 构建生产版本
```bash
npm run build
```

构建产物将输出到 `dist/` 目录

### 预览生产版本
```bash
npm run preview
```

## 📦 技术栈

- **前端框架**：React 19
- **构建工具**：Vite 7
- **样式**：纯CSS（无UI框架）
- **路由**：自定义路由系统（基于History API）

## 🔧 自定义配置

### 修改端口
编辑 `vite.config.js`：
```javascript
export default defineConfig({
  server: {
    port: 3000  // 修改为你想要的端口
  }
})
```

### 添加新页面
1. 在 `App.jsx` 中添加新的路由路径
2. 创建对应的页面组件
3. 在 `usePathname` hook 中添加有效路径
